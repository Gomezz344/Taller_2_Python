## 📚 Contenido por Sección

### Sección 1 — Manejo de Variables y Entrada de Datos
Introducción al uso de variables, tipos de datos primitivos y captura de información con `input()`. Se trabajan conversiones de tipos y operaciones básicas.

**Actividades:**
1. Declaración y uso de variables de diferentes tipos
2. Entrada de datos con `input()` y conversión de tipos
3. Operaciones aritméticas básicas
4. Concatenación y formateo de cadenas
5. Cálculos con datos ingresados por el usuario

---

### Sección 2 — Implementación de Condicionales
Uso de estructuras `if`, `elif` y `else` para tomar decisiones según condiciones evaluadas. Se trabajan operadores relacionales y lógicos.

**Actividades:**
1. Condicionales simples con `if/else`
2. Condicionales múltiples con `elif`
3. Operadores lógicos (`and`, `or`, `not`)
4. Validación de rangos y datos
5. Menú de opciones con condicionales

---

### Sección 3 — Uso de Ciclos `for` y `while`
Implementación de estructuras repetitivas para ejecutar bloques de código múltiples veces. Se trabajan ciclos con contadores, acumuladores y condiciones de parada.

**Actividades:**
1. Ciclo `for` con `range()`
2. Ciclo `while` con condición de entrada
3. Acumuladores y contadores
4. Ciclos anidados
5. Control de flujo con `break` y `continue`

---

### Sección 4 — Gestión de Listas y Diccionarios
Manejo de estructuras de datos para almacenar y organizar colecciones de información. Se trabajan listas, diccionarios y operaciones de búsqueda, inserción y eliminación.

**Actividades:**
1. Creación y recorrido de listas
2. Operaciones sobre listas (agregar, eliminar, buscar)
3. Listas sin duplicados con ciclos auxiliares
4. Creación y acceso a diccionarios
5. Lista de diccionarios para registros

---

### Sección 5 — Creación y Uso de Funciones
Modularización del código mediante funciones reutilizables. Se trabajan parámetros, retorno de valores, validaciones y recursividad.

**Actividades:**
1. Funciones con parámetros y retorno
2. Validación de datos dentro de funciones
3. Funciones auxiliares y menú principal
4. Recursividad (`factorial`)
5. Organización del código con múltiples funciones

---

### Sección 6 — Proyecto Final: Sistema de Gestión de Biblioteca
Proyecto integrador que aplica todos los conceptos del taller. Implementa un sistema completo de gestión bibliográfica con menú interactivo y operaciones CRUD.

**Funcionalidades:**
- Agregar, mostrar, buscar y eliminar libros
- Préstamo y devolución con control de disponibilidad
- Búsqueda por título o autor (coincidencias parciales)
- Estadísticas del catálogo
- Exportación del catálogo a archivo `.txt`

```bash
cd taller/seccion6
python biblioteca.py
```
